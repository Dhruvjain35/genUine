KEY FILES BREAKDOWN
===================

1. app/page.tsx (Main UI Component)
   - 3-step form: "your voice", "their profile", "your name"
   - Textarea inputs (3-5 user examples, up to 5)
   - Large target profile textarea
   - Name input field
   - Generate button with loading state
   - Results display: 2 message cards side-by-side
   - Copy-to-clipboard buttons with feedback
   - Rate limiting: 5 messages per session (React state)
   - Mobile responsive design
   - Brand colors: cream background, terracotta accents

2. app/api/generate-message/route.ts (API Endpoint)
   - POST endpoint receiving: userExamples[], targetProfile, userName
   - Validates minimum 2 examples + profile + name
   - Calls claude-sonnet-4-6 with custom system prompt
   - System prompt ensures AI learns user's exact writing style
   - Returns JSON: { messageA, messageB, styleNotes }
   - Error handling with warm error messages

3. app/globals.css (Styling)
   - Google Fonts: Plus Jakarta Sans (headings), DM Sans (body)
   - Custom animations: fadeIn, pulse-gentle
   - Focus states with terracotta border + shadow
   - Brand color variables
   - Smooth transitions throughout

4. app/layout.tsx (Layout)
   - Metadata: title, description, OpenGraph tags
   - Body styling: background (#FAF9F7), text color (#2D2D2D)
   - Fonts imported from Google

5. tailwind.config.ts (Tailwind Configuration)
   - Custom colors:
     * genuine-cream: #FAF9F7
     * genuine-terracotta: #C4784A
     * genuine-amber: #F2A922
     * genuine-dark: #2D2D2D
   - Custom fonts: jakarta, dm-sans
